IMAGES FOLDER - KODIKATHA KUMARAN CHARITABLE FORUM
=====================================================

Please place the following images in this folder:

1. kumaran-portrait.jpg
   - Portrait photo of Kodikatha Kumaran / Tirupur Kumaran
   - Used in: Heritage section portrait circle
   - Recommended size: 300×300px (square)

2. about-event.jpg
   - A group photo or event photo of the forum
   - Used in: About section image frame
   - Recommended size: 800×600px (landscape)

3. gallery-1.jpg to gallery-7.jpg
   - Event photos, award ceremonies, medical camps, cultural events
   - Used in: Gallery section grid
   - Recommended size: 600×600px each

4. og-image.jpg
   - Open Graph / Social share image
   - Used in: HTML meta tags for social sharing
   - Recommended size: 1200×630px

5. favicon.ico or favicon.png
   - Small icon for browser tab
   - Recommended size: 32×32px or 64×64px

NOTE:
- If images are not placed, the website will show styled
  placeholder icons and colors instead (fully functional).
- All images should be optimised/compressed before placing.
  Use tools like TinyPNG (tinypng.com) to compress.
- Supported formats: .jpg, .jpeg, .png, .webp
